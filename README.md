# Mecenate Mobile App - Запуск и настройка

## Как запустить

### 1. Установка зависимостей
```bash
npm install
```

### 2. Запуск приложения
```bash
# Запуск в режиме разработки
npm start

# Запуск на Android
npm run android

# Запуск на iOS
npm run ios

# Если проблемы с кешем
npx expo start --clear
```

## Требования

- Node.js 18+
- Expo CLI: `npm install -g expo-cli`
- Expo Go на физическом устройстве или эмулятор
